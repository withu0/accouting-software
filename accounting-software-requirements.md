# 1人社長向け 超シンプル会計ソフト 開発指南書

## 1. 開発コンセプト

本ソフトは、freeeやマネーフォワードのような網羅型会計ソフトではなく、1人社長・小規模法人向けに「簡単さ」を最優先した会計ソフトとして開発する。

対象ユーザーは、

* 1人社長
* 従業員が少ない法人
* 会計知識がない経営者
* 税理士へ渡すための記帳を自社で行いたい経営者

を想定する。

そのため、

* 勘定科目は最小限
* 会計知識を要求しない
* 銀行CSVを取り込むだけで記帳できる

ことを重視する。

---

# 2. システム構成

機能は以下の4つを中心とする。

① 銀行CSV取込（自動仕訳）

② 立替経費入力

③ 決算書出力

④ 臨時仕訳（振替伝票）

# 4. 銀行CSV取込（自動仕訳）

## 概要

法人口座のCSVを取り込み、自動で仕訳候補を生成する。

---

## 入金処理

法人口座への入金は、原則として売上として処理する。

仕訳

```
預金　100,000 / 売上高　100,000
```

※例外的な入金（借入金・資本金・役員貸付返済等）は振替伝票で処理する。

---

## 出金処理

法人口座からの出金は、ユーザーが経費科目を選択する。

例

```
旅費交通費　10,000 / 預金　10,000
```

```
通信費　5,000 / 預金　5,000
```

---

## 自動学習機能

摘要を学習し、次回以降は自動で科目候補を表示する。

例

| 摘要     | 自動提案  |
| ------ | ----- |
| Amazon | 消耗品費  |
| JR     | 旅費交通費 |
| NTT    | 通信費   |
| Google | 広告宣伝費 |
| 税務署    | 租税公課  |
| 日本年金機構 | 法定福利費 |

---

# 5. 立替経費入力

## 概要

社長個人が支払った経費を登録する機能。

---

## 入力項目

* 日付
* 金額
* 摘要
* 経費科目

---

## 自動生成仕訳

```
〇〇費　××× / 役員借入金　×××
```

例

```
会議費　5,000 / 役員借入金　5,000
```

```
広告宣伝費　30,000 / 役員借入金　30,000
```

利用者に借方・貸方を意識させない。

---

# 6. 臨時仕訳（振替伝票）

## 概要

通常処理で対応できない仕訳専用のメニュー。

メイン導線ではなく、

「その他」
↓
「臨時仕訳（振替伝票）」

として配置する。

---

## 用途

以下のような例外処理に使用する。

### 売掛金計上

```
売掛金　100,000 / 売上高　100,000
```

### 買掛金計上

```
消耗品費　50,000 / 買掛金　50,000
```

### 未払金計上

```
支払手数料　30,000 / 未払金　30,000
```

### 前払費用

```
前払費用　120,000 / 預金　120,000
```

### 減価償却

```
減価償却費　100,000 / 建物附属設備　100,000
```

### 法人税計上

```
法人税、住民税及び事業税　300,000
    / 未払法人税等　300,000
```

### 役員借入金返済

```
役員借入金　500,000 / 預金　500,000
```

---

## 入力項目

* 日付
* 借方科目
* 借方金額
* 貸方科目
* 貸方金額
* 摘要

# 12. 最重要設計思想

本ソフトは「会計ソフト」ではなく、

『1人社長が迷わず決算書を作るためのソフト』

として設計する。

そのため、

* 借方・貸方を極力見せない
* 勘定科目を増やさない
* 入力項目を最小化する
* 会計知識を要求しない

ことを徹底する。

利用者が行う作業は、

① 銀行CSVを取り込む

② 経費科目を選ぶ

③ 個人立替経費を入力する

④ 決算書を出力する

基本的にこの4ステップで完結する設計とする。
